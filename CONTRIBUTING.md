# Contributing

Issues and PRs welcome — templates, prompts, SDK improvements, and doc fixes especially.

- Templates go in `templates/` and must reference real `MODULE_TYPE` values and never
  hardcode config fields that aren't in `GET /bot-modules/definitions`.
- Keep everything truthful: no fabricated endpoints, fields, or capabilities.
- SDKs stay dependency-light.

Questions: https://t.me/botlaunch
